jsLSA Overview

Fundamentally, jsLSA functions as an integrator crafted for Second-Order nonlinear systems. It operates in two primary modes:

Static Mode:

In this mode, an array of 'Y' values is provided, and jsLSA returns a corresponding array of 'U' values. Each element in the array signifies the 'y' value of the function for each Δt.

Real-Time Mode:

This mode starts with an empty array. In each iteration, a value is supplied that corresponds to the 'y' value of the graph at that particular point. Visualize the graph dynamically evolving in real-time. The algorithm retains the previous output array, continually extending it (up to a defined position), and discarding the earliest elements to make room for the latest ones. The integration is then recalculated with the new data point, yielding an updated graph.

Second Order Systems

A transfer function is a mathematical representation of the relationship between the input and output of a linear time-invariant system. It is commonly used in the field of control theory and dynamic systems analysis. The transfer function is expressed as the ratio of the Laplace transforms of the system's output to its input. This representation is particularly useful for analyzing and designing control systems, as it provides insights into how the system responds to different frequencies and input signals. Transfer functions are a fundamental concept in understanding the behavior of dynamic systems in engineering and related fields

Second order differential equation

\[ a \frac{d^2y}{dt^2} + b \frac{dy}{dt} + cy = f(t) \]

For systems with amplification

Some systems have the feature to amplify the output signal. In this point of view the Differential Equation displays as follows:

\( \frac{d^2y}{dt^2} + 2\zeta\omega_n \frac{dy}{dt} + \omega_n^2y = Kx(t) \)

  • K: System gain or amplification.
  • ωn: Undamped natural frequency of the system.
  • ζ: Damping coefficient (damping ratio).

Laplacian interpretation of the system with Amplification

\( Y(s) = \frac{K}{\omega_n^2} \cdot \frac{\omega_n^2}{s(s + 2\zeta\omega_n)} X(s) \)

Clasification of Second Order Systems in function of damping (ζ)
  • For ζ < 1 (underdamped system), the system is oscillatory.
  • For ζ = 1 (critically damped system), the system response is the fastest without oscillations.
  • For ζ > 1 (overdamped system), the system is slow to respond, and there are no oscillations.
  • For ζ = 0 (undamped system), the system oscillates indefinitely without slowing down.

jsLSA works in the rank of underdamped oscillations

\[ 0 < \zeta < 1 \text{ (underdamped system)} \]

Dirac's Step Input Response

jsLSA integrates the sum of all responses to a Dirac step input with amplitude equal to the function's amplitude.

Laplacian representation of a Dirac's Step:

\( R(s)=\frac{1}{s} \)

For underdamped rank:

\(H(s) = \frac{\omega_n^2}{s(s + 2\zeta\omega_n)} \)

Inversing the Laplacian function to time domain:

\(y(t) = 1 - e^{-\delta \omega_n t} \left(\cos(\sqrt{1-\delta^2}\omega_n t) + \frac{\delta}{\sqrt{1-\delta^2}}\sin(\sqrt{1-\delta^2}\omega_n t)\right)\)

  • \(y(t)\) is the step response in time.
  • \(\delta\) is the damping coefficient (damping ratio).
  • \(\omega_n\) is the undamped natural frequency of the system.
  • \(\omega_d\) is the damped angular frequency, given by \(\omega_d = \omega_n \sqrt{1-\delta^2}\).
Reducing the response equation

\(y(t) = 1 - e^{-\delta \omega_n t} \left(\cos(\omega_d t) + \frac{\delta}{\sqrt{1-\delta^2}}\sin(\omega_d t)\right)\)

Calculating a non-unitary step in function of u(t)

\(y(t) = u(t) - u(t) e^{-\delta \omega_n t} \left(\cos(\omega_d t) + \frac{\delta}{\sqrt{1-\delta^2}}\sin(\omega_d t)\right)\)

Convolution of Signals

jsLSA works convoluting the signals following the Convolution Theorem of Signals. The convolucion of signas \( f*g \) produces a new signal \( h \) that indicates how one wave changes respect to the other one.

Convolution definition for a definited time space:

\( (f \ast g_{T})(t) \equiv \int _{t_{0}}^{t_{0}+T}\left[\sum _{k=-\infty }^{\infty }f(\tau +kT)\right]g_{T}(t-\tau )\,d\tau \)

Discretized convolution as Sum of signals:

\( (f \ast g)[n] = \sum_{m=-\infty}^{\infty} f[m] \cdot g[n-m] \)

Accuracy

jsLSA has been meticulously crafted with a focus on accuracy and efficiency to provide a fast and reliable integrator for both development and research purposes. Our aim is not just to approximate, but to deliver results that closely align with the ideal case.

Stringent accuracy standards:

To validate and ensure the accuracy of results, the logic has undergone rigorous testing against high-tech, quality software. A Matlab Suite guarantees that the code produces meaningful and precise responses.
